# AWS Agentic AI Architecture — Cheat Sheet
A compact, interactive reference card covering the full AWS Bedrock agentic AI stack — built for developers and architects preparing for the AWS Agentic AI Micro-credential or working on production agentic systems.

Covers foundation model selection (Claude, Titan, Llama, Mistral, Cohere), core Bedrock services (Agents, Knowledge Bases, Guardrails, Flows, Inline Agents, Multi-Agent Collaboration), memory strategies, vector store options, action execution patterns, and observability tooling — with a quick-decision table mapping common needs to the right AWS service.

Click any card to expand when-to-use guidance. Fully self-contained HTML with light/dark mode support and mobile-responsive layout. No dependencies, no build step — works offline and deploys directly to GitHub Pages.
